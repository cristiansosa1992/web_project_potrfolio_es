# Portafolio

https://cristiansosa1992.github.io/web_project_portfolio_es/

Durante el desarrollo de este proyecto he fortalecido mis habilidades en HTML y CSS, enfocándome en la construcción de interfaces claras, organizadas y adaptables a diferentes dispositivos.
El proyecto está diseñado para funcionar correctamente en desktop, tablet y mobile, por lo que he trabajado especialmente en diseño responsivo.
A lo largo del proceso he aplicado conceptos como Flexbox y Grid para la organización de layouts, uso de unidades responsivas como rem, vw y vh, manejo de media queries, adaptación de imágenes y proporciones con aspect-ratio, así como el control de espaciados, gap, line-height y pseudo-clases como :hover.
Este proyecto me ha permitido comprender mejor cómo construir interfaces web que se adapten correctamente a distintos tamaños de pantalla, manteniendo una estructura limpia y estilos consistentes.
